Comparative Evolutionary Analysis of the BRCA1 Gene Across Mammalian Species

Overview

This project investigates the evolutionary conservation of the BRCA1 gene across five mammalian species using freely available bioinformatics tools. The study compares BRCA1 mRNA sequences through pairwise sequence alignment, multiple sequence alignment, and phylogenetic analysis to understand evolutionary relationships among mammals.

Objectives

- Retrieve BRCA1 mRNA sequences from the NCBI database.
- Compare BRCA1 sequences using BLASTN.
- Perform multiple sequence alignment using MUSCLE.
- Construct a Neighbor-Joining phylogenetic tree using MEGA11.
- Evaluate evolutionary relationships among selected mammalian species.

Selected Species

- Homo sapiens (Human)
- Macaca mulatta (Rhesus macaque)
- Canis lupus familiaris (Dog)
- Mus musculus (Mouse)
- Rattus norvegicus (Rat)

Bioinformatics Tools

- NCBI Gene
- NCBI BLASTN
- MUSCLE
- MEGA11
- Microsoft Excel

Workflow

1. Sequence retrieval from NCBI
2. Download sequences in FASTA format
3. Pairwise sequence comparison using BLASTN
4. Multiple sequence alignment using MUSCLE
5. Phylogenetic tree construction using Neighbor-Joining
6. Bootstrap analysis (500 replicates)
7. Interpretation of evolutionary relationships

Major Findings

- BRCA1 is highly conserved among mammalian species.
- Human and rhesus macaque showed the highest sequence similarity.
- Mouse and rat formed a separate rodent clade.
- Dog occupied an intermediate evolutionary position.
- Bootstrap analysis supported the inferred phylogenetic relationships.
